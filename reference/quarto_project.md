# Create a Custom Quarto Project

**Deprecated.** Use
[`init()`](https://www.kyleGrealis.com/froggeR/reference/init.md)
instead.

## Usage

``` r
quarto_project(name, path = here::here(), example = TRUE)
```

## Arguments

- name:

  Character. Ignored.

- path:

  Character. Ignored.

- example:

  Logical. Ignored.

## Value

Does not return. Always errors with a deprecation message.
